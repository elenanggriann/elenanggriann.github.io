Website Sistem Informasi BEM FMIPA UM

Project information

This project is part of the final project of the Database course which requires students to make a basic website/application that focusing on the "database system". The students are required to present at least 3 tables from the database into the developed website.

The website Sistem Informasi BEM FMIPA UM has 7 menus, namely Dashboard, Anggota, Bidang, Program Kerja, Kepanitiaan, Pengaturan, and Keluar. My contribution in this team project is to create UI/UX design and apply it to the front-end design using HTML, CSS, and JavaScript.
